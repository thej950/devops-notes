Create a VM on GCP
----------------------
1. Enable Compute Engine API 
2. Create VM 

	Machine Configurations
	OS and Storage
	Data Protection
	Networking
	Observability
	Security
	Advanced

	Compute Engine->VM Instances->click on create Instance->
		Name: Jenkins-machine
		Labels: N/A
		Region: (Go with Defaults)
		Machine configuration: Go with General Purpose 
			Series: Ec2
			Machine type: ec2-micro, 2CPU, 1GB RAM
		Confidential VM server: N/A
		Container: N/A
		Boot disk:  (Go with default)
		Identity and API access: (Go with Default)
		Access Scopes: (Allow Default Access)
		Firewall: (Based usage set required details )
		
		- any Additional Info Go with Default
