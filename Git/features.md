### Sample Interview Answer

Git is a Distributed Version Control System (DVCS) that enables efficient team collaboration. It supports branching, merging, staging, and provides strong data integrity using SHA-1 hashes. Git is fast, secure, open-source, and works offline while integrating seamlessly with platforms like GitHub.
